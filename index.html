Project Overview

☍

DNS & Domain

Domain resolution via GitHub Pages subdomain. DNS maps human-readable hostnames to IP addresses using recursive queries through the DNS hierarchy.

♺

IP Addressing

IPv4 and IPv6 resolution demonstrated through dig and nslookup commands, showing how the site resolves to GitHub's content delivery network.

⇄

HTTP/HTTPS

HTTPS enforced via GitHub Pages with TLS certificates. HTTP headers, status codes, and the TLS handshake process are documented and explained.

⚠

Security

HTTPS enforcement ensures encrypted communication. Security headers and certificate validation protect data integrity and authenticity.
