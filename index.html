How Networking Makes This Site Work

Project Overview

01

DNS Resolution

This site's domain resolves through the DNS hierarchy — root servers → TLD nameservers → authoritative nameservers — returning IP addresses so browsers can connect.

nslookup · dig · A/AAAA records
02

IP Addressing

The domain resolves to both IPv4 and IPv6 addresses on Vercel's edge network, demonstrating dual-stack connectivity across their global CDN.

IPv4 · IPv6 · Dual-stack
03

HTTP/HTTPS Protocols

All traffic is served over HTTPS (port 443) with TLS encryption. HTTP requests are redirected. Response headers and status codes are documented with evidence.

TLS 1.3 · 200 OK · 308 Redirect
04

Security

HTTPS enforced automatically via Vercel. Security headers including HSTS, X-Content-Type-Options, and X-Frame-Options protect against MITM, XSS, and clickjacking.

HSTS · Certificate · Headers
